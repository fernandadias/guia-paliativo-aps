# Áudio

Coloque aqui a música de fundo do guia:

- **`music-bg.mp3`** — toca em loop durante todo o guia, com volume suave.

A música só começa quando a pessoa clica em "Começar" na abertura (o navegador
exige um gesto do usuário para liberar áudio). Há um botão na barra do guia para
ligar/desligar a música a qualquer momento.
